# 开发测试验收

## 验收目标

- 项目可安装依赖并完成构建
- 核心单元测试全部通过
- 多节点集成测试通过
- 可启动 Web API 并返回节点状态
- 配置、加密、冲突解决、同步写库行为可验证

## 验收清单

### 功能

- 配置文件自动生成与读取
- 文件内容加密与解密正确
- 冲突解决规则稳定
- 本地文件变化可生成 CID 与版本记录
- Web API 可返回状态、节点、目录和文件列表

### 非功能

- TypeScript 编译无错误
- 测试命令全部通过
- 多节点集成验证可单独执行
- 启动命令可运行

## 集成验证命令

- 全量验证：npm test
- 多节点专项验证：npm run test:integration

当前项目已补充综合多节点内存网络集成测试，覆盖：

- 节点发现与 announce 广播
- 双向信任建立与同步门禁
- 建立互信后的实时文件同步
- 互信建立后的历史状态回补
- 加密内容在远端正确拉取并解密
- 删除事件传播与远端落盘删除
- include/exclude 过滤规则
- 冲突场景下较新版本保留
- Web API 认证与状态/节点/文件查询

## 后续扩展

- 接入真实 Kubo 多节点联调
- 补充浏览器端交互页面
- 增加 SSE 推送与更完整的文件浏览界面
